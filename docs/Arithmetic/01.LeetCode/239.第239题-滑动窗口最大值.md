### [题目描述](https://leetcode-cn.com/problems/sliding-window-maximum/comments/)

给你一个整数数组 <span style="background: #eee; color: #666;">nums</span> ，有一个大小为  <span style="background: #eee; color: #666;">k</span>  的滑动窗口从数组的最左侧移动到数组的最右侧。你只可以看到在滑动窗口内的 <span style="background: #eee; color: #666;">k</span>  个数字。滑动窗口每次只向右移动一位。

返回滑动窗口中的最大值。

<!-- more -->

**示例 1:**

```
输入：nums = [1,3,-1,-3,5,3,6,7], k = 3
输出：[3,3,5,5,6,7]
解释：
滑动窗口的位置                最大值
---------------               -----
[1  3  -1] -3  5  3  6  7       3
 1 [3  -1  -3] 5  3  6  7       3
 1  3 [-1  -3  5] 3  6  7       5
 1  3  -1 [-3  5  3] 6  7       5
 1  3  -1  -3 [5  3  6] 7       6
 1  3  -1  -3  5 [3  6  7]      7
```

**示例 2:**

```
输入：nums = [1], k = 1
输出：[1]
```

**示例 3:**

```
输入：nums = [1,-1], k = 1
输出：[1,-1]
```

**示例 4:**

```
输入：nums = [9,11], k = 2
输出：[11]
```

**示例 5:**

```
输入：nums = [4,-2], k = 2
输出：[4]
```

**示例 6:**

```
输入：nums = [7,2,4], k = 2
输出：[7,4]
```

**注意:**

- <span style="background: #eee; color: #666;">1 <= nums.length <= 105</span>
- <span style="background: #eee; color: #666;">-104 <= nums[i] <= 104</span>
- <span style="background: #eee; color: #666;">1 <= k <= nums.length</span>

### 单调队列

- 维护一个单调队列 <span style="background: #eee; color: #666;">queue</span> ，用来存滑动窗口**最大值**的 <span style="background: #eee; color: #666;">index</span> **索引**
- 建立一个新数组 <span style="background: #eee; color: #666;">newArr</span> 用来存滑动窗口最大值
- 每次窗口滑动时候，判断 <span style="background: #eee; color: #666;">queue</span> 队列的队头是**否在窗口范围内**，如果不在，队头出队
- 再就继续

```JavaScript
/**
 * @param {number[]} nums
 * @param {number} k
 * @return {number[]}
 */
const maxSlidingWindow = (nums, k) => {
  const len = nums.length
  // queue 队列是用来获取滑动窗口最大值的index的
  const queue = []

  if (k === 1 && len !== 0) return nums

  // 长度为 k 的滑动窗口，滑动前的最大值索引队列
  for (let i = 0; i < len && i < k; i++) {
    while (queue.length && nums[i] >= nums[queue[queue.length - 1]]) {
      queue.pop()
    }
    queue.push(i)
  }

  const newArr = [nums[queue[0]]]

  // 开始滑动
  for (let i = k; i < len; i++) {
    // 如果 queue 队头不是最大值，就踢掉😂
    while (queue.length && nums[i] >= nums[queue[queue.length - 1]]) {
      queue.pop()
    }
    queue.push(i)
    // 以下这一步是维护窗口内最大值是在窗口范围的
    while (queue[0] <= i - k) {
      queue.shift()
    }
    // 添加队列中第一个，也就是最大值
    newArr.push(nums[queue[0]])
  }

  return newArr
}
```
