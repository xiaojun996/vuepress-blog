### [题目描述](https://leetcode-cn.com/problems/move-zeroes/)

给定一个数组 <span style="background: #ddd; color: #666; padding: 3px 5px; border-radius: 2px;">nums</span>，编写一个函数将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。

**说明:**

- 必须在原数组上操作，不能拷贝额外的数组。
- 尽量减少操作次数。

**示例:**

```
输入: [0,1,0,3,12]
输出: [1,3,12,0,0]
```

### 双指针

- tick 代表次数，i 代表坐标
- 简单题就不解释了 =。=
- 这题 和 [加油站](https://xiaojun996.top/pages/7a4af3/) 很类似

```JavaScript
/**
 * @param {number[]} nums
 * @return {void} Do not return anything, modify nums in-place instead.
 */
const moveZeroes = (nums) => {
  let len = nums.length
  for (let i = tick = 0; tick <= len; i++ , tick++) {
    if (nums[i] === 0) {
      nums.splice(i, 1)
      nums.push(0)
      i--
    }
  }

  return nums
}
```
