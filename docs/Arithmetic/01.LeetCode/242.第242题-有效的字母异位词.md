### [题目描述](https://leetcode-cn.com/problems/valid-anagram/)

给定两个字符串 s 和 t ，编写一个函数来判断 t 是否是 s 的字母异位词。

**示例 1:**

```
输入: s = "anagram", t = "nagaram"
输出: true
```

<!-- more -->

**示例 2:**

```
输入: s = "rat", t = "car"
输出: false
```

**说明:**
你可以假设字符串只包含小写字母。

**进阶:**
如果输入字符串包含 unicode 字符怎么办？你能否调整你的解法来应对这种情况？

## sort 排序

### 思路

- 看代码就懂了～

### 代码

```JavaScript
/**
 * @param {string} s
 * @param {string} t
 * @return {boolean}
 */
const isAnagram = (s, t) => {
  s = [...s].sort().join('')
  t = [...t].sort().join('')

  return s === t
}
```

## hash 表

### 思路

通过哈希表存取字符次数

- 遍历字符串 s 和字谜 t
  - s[i] 加一
  - t[i] 减一
- 最后判断每个值是否都是 0

### 代码

```JavaScript
/**
 * @param {string} s
 * @param {string} t
 * @return {boolean}
 */
const isAnagram = (s, t) => {
  if (s.length !== t.length) return false

  const hash = new Map()

  let i = s.length - 1

  while (i >= 0) {
    hash.set(s[i], (hash.get(s[i]) || 0) + 1)
    hash.set(t[i], (hash.get(t[i]) || 0) - 1)
    i--
  }

  return [...hash.entries()].every(v => v[1] === 0)
}
```
