通过 Node 中运行 process.arch 查看

```
~ which zsh
/bin/zsh
~ /bin/zsh --version
zsh 5.8 (x86_64-apple-darwin20.0)
~ arch
arm64
~ node
Welcome to Node.js v15.11.0.
Type ".help" for more information.
> process.arch
'x64'
>
```
