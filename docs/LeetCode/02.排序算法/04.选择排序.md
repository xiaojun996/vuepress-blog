# 选择排序

JavaScript 实现的选择排序算法

<!-- more -->

### 代码

```JavaScript
/**
 * @param {Array} arr
 * @return {Array}
 */
const selectionSort = arr => {
  const len = arr.length
  let min
  for (let i = 0; i < len; i++) {
    min = i
    for (let j = i + 1; j < len; j++) {
      if (arr[j] < arr[min]) {
        // 保存最小的数的小数
        min = j
      }
    }
    ;[arr[i], arr[min]] = [arr[min], arr[i]]
  }

  return arr
}

```
