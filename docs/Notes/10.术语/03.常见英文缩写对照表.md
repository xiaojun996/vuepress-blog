|    缩写    |                英文                |    中文    |
| :--------: | :--------------------------------: | :--------: |
| **TL;DR:** |       too long; didn't read        |  太长不读  |
|  **ASAP**  |        as soon as possible         |    尽快    |
|  **RIP**   |           rest in peace            |  逝者安息  |
|  **e.g.**  | for example(拉丁文 exempli gratia) |  举个例子  |
|  **etc**   |    and so on(拉丁文 et cetera)     |    等等    |
|  **BTW**   |             by the way             | 顺便说一下 |
